% THE CCI-MOBILE VOCODER

Step 1: Run "Vocoder_Design_gui" MATLAB function.

Step 2: Press Load Audio file (Select .wav file from your directory).

Step 3: Select Number of Filters (default: 4).

Step 4: Select Filter type (default: FIR).

Step 5: Select Vocoder Method (default: Noise band).

Step 6: Press 'Update'.

Step 7: Listen Original and vocoded signal by pressing "Play Original Signal"/"Play Vocoded Signal" button.

Step 8: If you reselect any one of Step e, 4 or 5, follow step 6.
