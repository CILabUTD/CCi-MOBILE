Step 1: Push "Load Audio file" button to load audio file from 'Data' folder.

Step 2: Select Noise type by clicking any butten from 'Noise Selectoin' panel.

Step 3: Select SNR type by clicking any butten from 'SNR' panel.

Step 4: Select Enhancement Method by clicking any butten from 'Enhancement Method' panel.

Step 5: Push update button. You will see results.

Step 6: If you update any one from Step 2-4, follow step 5 again.
